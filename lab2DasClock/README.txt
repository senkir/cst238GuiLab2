GUI Lab 2 README

contained in this series of directories is the lab 2 project for OIT.

you can open the project in XCode through lab2DasClock.xcodeproj
This lab is based in Objective-C with Cocoa.  

DOCUMENTATION.txt contains project notes throughout the building of the lab.

this project is also available at github by visiting http://github.com/senkir/cst238GuiLab2

As for navigation once the app is running, the user can use the dropdown inside the main window to select which form of the clock to view.  They then select 'submit' and the view will update to the appropriate clock.
